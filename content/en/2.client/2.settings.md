# Client Settings (UI)

All changes in the client settings UI are auto-saved to `Settings.json`.

## General

- `Language` — client UI language.
- `Notification Dismiss` — notification auto-hide timeout (ms).
- `Hide Server Addresses` — hides server addresses in UI.
- `Disable Notifications` — disables local notifications completely.

## Appearance

- `Theme` — selected UI theme.
- `Background Image` — app background image.

## Input

- `Input Devices` — microphone/capture device.
- `Input Volume` — input gain (`0..2`).
- `Microphone Sensitivity` — voice activity threshold (`0..1`).
- `Denoisers` — noise suppression.
- `Automatic Gain Controllers` — automatic gain control.
- `Echo Cancelers` — echo cancellation.
- `Microphone Test` — local mic test + activity indicator.

## Output

- `Output Devices` — playback device.
- `Output Volume` — output volume (`0..2`).
- `Audio Clippers` — signal clipper/limiter type.
- `Test Output` — output test on selected device.

## Network

- `Positioning Type` — positioning mode (must match server).
- `McWss Listen Ip` — local address for McWss link.
- `McWss Host Port` — port for McWss link.

## HotKey

By default:

- `Mute` — `LeftControl + LeftShift + M`
- `Deafen` — `LeftControl + LeftShift + D`

## Advanced

- `Trigger GC` — force-run garbage collector.
- `Crash` — app crash test (debugging/logging verification).
