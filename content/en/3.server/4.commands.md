# Server Commands

While the server is running, console commands are available.

## Basic

- `list [--clientsOnly] [--limit N]` — list entities/clients.
- `stop` — stop the server.
- `kick <id>` — kick client.

## Client State Management

- `mute <id>`
- `unmute <id>`
- `deafen <id>`
- `undeafen <id>`

## Entity Data Management

- `setname <id> <value>`
- `settitle <id> <value>`
- `setdescription <id> <value>`
- `setposition <id> <x> <y> <z>`
- `setworldid <id> <value>`

## Practical Scenario

```text
list --clientsOnly
setworldid 12 spawn_world
setposition 12 100 64 100
mute 15
```

If ID is not found or entity is not a client, server returns a diagnostic error.
