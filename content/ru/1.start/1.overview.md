# Обзор

VoiceCraft состоит из трёх основных частей:

1. Клиентское приложение (`VoiceCraft.Client`) на Avalonia/.NET.
2. Сервер (`VoiceCraft.Server`) для голосового транспорта и синхронизации.
3. Minecraft-интеграция через аддоны (`McHttp` и `McWss`).

## Как это работает

1. Клиент подключается к `VoiceCraft.Server` по UDP (дефолт `9050`).
2. Сервер синхронизирует состояние игроков и применяет аудио-эффекты (proximity, muffle, echo и т.д.).
3. Minecraft-часть (через McHttp/McWss) связывает игровые события и голосовой слой.

## Поддерживаемые платформы клиента

- Windows (`x86`, `x64`, `arm64`)
- Linux (`x64`, `arm32`, `arm64`)
- macOS (`x64`, `arm64`)
- Android (`arm32`, `arm64`)
- iOS (`arm64`, в виде `.ipa`)
