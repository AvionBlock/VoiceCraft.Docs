# Быстрый старт

## 1) Скачайте сервер

1. Откройте [последний релиз](https://github.com/AvionBlock/VoiceCraft/releases/latest).
2. Скачайте архив под вашу платформу:
   - `VoiceCraft.Server.Windows.x64.zip` (или `x86`/`arm64`)
   - `VoiceCraft.Server.Linux.x64.zip` (или `arm`/`arm64`)

## 2) Запустите сервер

### Windows

```powershell
./VoiceCraft.Server.exe
```

### Linux

```bash
chmod +x ./VoiceCraft.Server
./VoiceCraft.Server
```

После первого запуска будет создан файл `config/ServerProperties.json`.

## 2.1) Сразу смените токены

Перед подключением Minecraft обязательно замените:

- `McHttpConfig.LoginToken`
- `McWssConfig.LoginToken`

в `config/ServerProperties.json` на ваши уникальные значения.

## 3) Скачайте клиент

С релизной страницы скачайте пакет под платформу:

- Windows: `VoiceCraft.Client.Windows.<arch>.zip`
- Linux: `VoiceCraft.Client.Linux.<arch>.zip`
- macOS: `VoiceCraft.Client.MacOS.<arch>.dmg` или `.pkg`
- Android: `VoiceCraft.Client.Android.<arch>.zip` (внутри APK)
- iOS: `VoiceCraft.Client.iOS.arm64.ipa`

## 4) Добавьте сервер в клиенте

1. Откройте клиент.
2. Добавьте сервер в списке (`IP` + `Port`).
3. Обычно это `127.0.0.1:9050`, если всё на одной машине.

## 5) Подключите Minecraft слой

- Для Bedrock Dedicated Server используйте [McHttp](/ru/minecraft/mchttp-bds).
- Для singleplayer world используйте [McWss](/ru/minecraft/mcwss-singleplayer).

## Проверка

Если всё настроено корректно:

- сервер показывает запуск `VoiceCraft`,
- клиент подключается без ошибок,
- при входе игроков в мир появляется голосовая синхронизация.
