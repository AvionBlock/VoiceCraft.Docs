# McHttp для Bedrock Dedicated Server

`McHttp` — рекомендуемый вариант интеграции для BDS.

## Что нужно

1. Запущенный `VoiceCraft.Server` с включенным `McHttpConfig.Enabled: true`.
2. Аддон `VoiceCraft.Addon.Core.McHttp.zip` из релизов.
3. BDS сервер с включенными beta/script API (в зависимости от версии Bedrock).

## Установка аддона

1. Распакуйте `VoiceCraft.Addon.Core.McHttp.zip`.
2. Положите `RP` в `<MCServer>/resource_packs/`.
3. Положите `BP` в `<MCServer>/behavior_packs/`.

## Права модулей

Откройте `<MCServer>/config/default/permissions.json` и убедитесь, что есть:

```json
{
  "allowed_modules": [
    "@minecraft/server-gametest",
    "@minecraft/server",
    "@minecraft/server-ui",
    "@minecraft/server-admin",
    "@minecraft/server-editor",
    "@minecraft/server-net"
  ]
}
```

## Подключение pack'ов к миру

В `<MCServer>/worlds/<YourWorld>/world_behavior_packs.json` добавьте:

```json
{
  "pack_id": "71ebb3ba-e9db-4546-9520-05f20b17dcb6",
  "version": [1, 4, 0]
}
```

В `world_resource_packs.json` добавьте:

```json
{
  "pack_id": "30b512be-77d1-4a61-bdb7-6c2f4062f889",
  "version": [1, 4, 0]
}
```

## Подключение из игры

В мире выполните:

```text
/voicecraft:vcconnect "http://<VOICECRAFT_HOST>:<PORT>" <LOGIN_TOKEN>
```

Пример:

```text
/voicecraft:vcconnect "http://127.0.0.1:9050" e4ad1f7e-4f90-4b21-bc15-6febe580bf1c
```

`LOGIN_TOKEN` берите из `McHttpConfig.LoginToken` в `ServerProperties.json`.

Если в вашей версии Bedrock разрешён short alias, команда может работать и как `/vcconnect`, но безопасный вариант — использовать namespaced форму `voicecraft:vcconnect`.

## Частые проблемы

- `HttpListenerException` на Windows: может понадобиться `netsh http add iplisten 127.0.0.1`.
- В контейнерах/VM используйте `Hostname` вида `http://0.0.0.0:9050/`.
- Проверяйте, что порт доступен и не занят другим процессом.
