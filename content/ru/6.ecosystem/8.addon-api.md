# Addon API

`VoiceCraft.Addon` даёт куда более широкую script-driven McApi поверхность, чем просто `vcbind`.

Эта страница предназначена для разработчиков аддонов и world automation.

## Основная API-поверхность

Addon-side API предоставляет:

- lifecycle соединения
- отправку и приём пакетов
- создание и удаление сущностей
- обновления world ID, position, rotation, mute, deafen и bitmask-полей
- effect updates
- audio-received events

## Высокоуровневые события

В текущем API есть:

- `OnConnected`
- `OnDisconnected`
- `OnPlayerBind`
- `OnPlayerUnbind`
- `OnPacket`

Системные script events включают:

- `voicecraft:onConnected`
- `voicecraft:onDisconnected`
- `voicecraft:onPlayerBind`
- `voicecraft:onPlayerUnbind`
- `voicecraft:onPacket`
- `voicecraft:sendPacket`

## Что можно строить на этом API

- auto-bind по ролям, командам или тегам
- кастомный bind UI
- effect presets по биомам и зонам
- region-based world ID remapping
- staff moderation tools
- scripted NPC или fake-entity voice logic
