# Первый запуск сервера

## Что происходит при первом запуске

При старте сервер ищет `ServerProperties.json` в текущей директории и поддиректориях.

Если файл не найден, сервер автоматически создаёт:

- папку `config/`
- файл `config/ServerProperties.json`

## Аргументы запуска

- `--exit-on-invalid-properties`
  Сервер завершится, если конфиг повреждён или невалиден.
- `--language <culture>`
  Переопределяет язык запуска, например `en-US`, `nl-NL`, `zh-CN`, `zh-TW`.

Пример:

```bash
./VoiceCraft.Server --exit-on-invalid-properties --language en-US
```

## Порты по умолчанию

- VoiceCraft UDP: `9050`
- McHttp: `http://127.0.0.1:9050/`
- McWss: `ws://127.0.0.1:9051/`

## Быстрый чек-лист перед продом

1. Измените `LoginToken` для `McHttpConfig` и `McWssConfig`.
2. Проверьте firewall/NAT и проброс портов.
3. Убедитесь, что `PositioningType` согласован с клиентами.
4. Ограничьте `MaxClients` под ресурсы вашего хоста.
