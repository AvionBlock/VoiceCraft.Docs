# Настройки клиента (UI)

Все изменения в настройках клиента сохраняются автоматически в `Settings.json`.

## General

- `Language` — язык интерфейса клиента.
- `Notification Dismiss` — время автоскрытия уведомлений (мс).
- `Hide Server Addresses` — скрывает адреса серверов в UI.
- `Disable Notifications` — полностью выключает локальные уведомления.

## Appearance

- `Theme` — выбранная тема UI.
- `Background Image` — фон приложения.

## Input

- `Input Devices` — микрофон/устройство захвата.
- `Input Volume` — усиление входного сигнала (`0..2`).
- `Microphone Sensitivity` — порог голосовой активности (`0..1`).
- `Denoisers` — шумоподавление.
- `Automatic Gain Controllers` — автоподстройка усиления.
- `Echo Cancelers` — эхо-компенсация.
- `Microphone Test` — локальный тест микрофона + индикатор активности.

## Output

- `Output Devices` — устройство воспроизведения.
- `Output Volume` — громкость выхода (`0..2`).
- `Audio Clippers` — тип clipper/ограничителя сигнала.
- `Test Output` — тест звука на выбранном устройстве.

## Network

- `Positioning Type` — режим позиционирования (должен совпадать с сервером).
- `McWss Listen Ip` — адрес для локальной McWss-связки.
- `McWss Host Port` — порт McWss-связки.

## HotKey

По умолчанию:

- `Mute` — `LeftControl + LeftShift + M`
- `Deafen` — `LeftControl + LeftShift + D`

## Advanced

- `Trigger GC` — принудительный запуск garbage collector.
- `Crash` — тест падения приложения (для дебага и проверки логирования).
